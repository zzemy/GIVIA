# Homepage Polish Pass Design

## Summary

This polish pass refines the already-upgraded homepage without changing the chosen Luxury Advisor System direction. It keeps the five capability cards and focuses on interaction clarity, hierarchy cleanup, and removal of the last dashboard-style remnants.

## Approved refinements

1. Keep **five** capability cards in the insight carousel.
2. Split hero CTA behavior so the primary CTA enters the workflow and the secondary CTA scrolls to the explanation/insight section.
3. Update the capability heading copy to match the five-card reality.
4. Restyle the workflow progress indicator to align with the premium navy + champagne system.
5. Compress the analysis configuration area so advanced enhancement controls are hidden by default and no longer dominate the main task flow.
6. Reduce redundancy in the country summary card by removing repeated controls and turning the footer into a lighter contextual status note.

## Success criteria

- Hero CTAs no longer perform the same action.
- Capability copy and actual card count are consistent.
- Workflow progress looks like part of the new premium system rather than an older cyan dashboard.
- Analysis step feels less like a configuration console.
- Country summary reads as lightweight confirmation rather than repeated form content.
